# pre-cognitive-neuronal-core

**Skill Primario Pre-Cognitivo**

Este es el punto de partida neuronal obligatorio del ecosistema soberano.

## Principios
- Predictive Coding jerárquico
- Active Inference
- Sparse activation
- Flujo ida y vuelta (predicción → error → actualización)

Este skill debe activarse primero en cualquier proceso cognitivo complejo.
