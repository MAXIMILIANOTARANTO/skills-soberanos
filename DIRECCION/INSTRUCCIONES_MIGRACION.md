# Instrucciones de Migración del Ecosistema Soberano

Este documento explica cómo mantener y expandir el ecosistema de skills soberanos en GitHub.

## Estructura actual
- Orquestador con inmunidad narrativa automática
- Núcleo pre-cognitivo como punto de partida
- Módulos cognitivos especializados
- Memoria externa con estructura blockchange
- Protección contra inyecciones narrativas

## Flujo de trabajo recomendado
1. Crear nuevo skill en carpeta correspondiente
2. Actualizar MANIFEST.md
3. Probar integración con orquestador-soberano
4. Subir con commit claro
5. Registrar en memoria persistente si es relevante
